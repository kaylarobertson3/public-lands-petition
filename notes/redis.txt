ABOUT REDIS =======================

- Doesn't store data in tables. It has various structures to keep your data. Most basic way is key, value pairs.

- It's a database ish thing for things that you don't want to save forever.

- It keeps data in memory, not disk

-Redis is good for caching


- updated path to redis srs, can run redic-server from anywhere to start redis server
